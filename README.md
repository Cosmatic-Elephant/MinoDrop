# Mino Drop

MinoDrop은 개인적인 즐거움과 실험을 목적으로 만들어진 비영리 팬 프로젝트입니다.  
Tetris® 및 관련 상표권은 The Tetris Company, LLC에 있으며, 본 프로젝트는 The Tetris Company와 공식적인 제휴 또는 승인 관계에 있지 않습니다.

---

## 페이지 이용 안내

### 메인
게임을 플레이하는 페이지입니다.  
설정 메뉴에서 필드 크기, 레벨, 조작감(DAS/ARR) 등 게임 환경을 조정할 수 있으며, 사용할 미노 팩과 킥테이블을 선택할 수 있습니다.

### 내 목록
제작한 미노 팩과 킥테이블을 관리하는 페이지입니다.  
팩 및 킥테이블을 코드로 내보내거나 불러올 수 있어 다른 기기나 사람과 공유할 수 있습니다.

### 미노 팩 커스터마이징
미노 팩 안의 미노를 직접 그려서 만드는 페이지입니다.  
캔버스에 셀을 칠해 미노의 모양을 정의하고, 이름과 색상을 지정해 팩에 추가할 수 있습니다.

### 킥테이블 커스터마이징
미노별 회전 오프셋(킥)을 직접 편집하는 페이지입니다.  
각 회전 방향에 대해 시도할 오프셋을 캔버스로 지정하며, 미러링 등의 도구로 편집을 보조합니다.
